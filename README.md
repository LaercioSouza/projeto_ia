# projeto_ia
Modelo de inteligência artificial para calcular as principais regiões com maiores indices de evasão escolar.
Para executar o projeto siga os passos:
1) Extraia os arquivos
2) Instale a biblioteca no prompt de comando: pip install scikit-learn pandas numpy matplotlib
3) Execute o arquivo index.py
4) Ao executar, vai aparecer gráficos com dados do algoritmo, serão exibidos um por um, feche clicando no x para ver o próximo
